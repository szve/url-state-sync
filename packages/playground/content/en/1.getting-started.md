# Getting started

`url-state-sync` is a minimal, type-safe store that keeps your state in memory and
syncs it to the URL query — **you decide when it writes, and values keep their declared types.**

## Install

```bash
pnpm add url-state-sync
```

## Usage

```ts
import { createURLStore } from 'url-state-sync'

const store = createURLStore({
  keyword: '',
  page: 1,
  tags: [] as string[],
})

store.set({ keyword: 'hello' })      // memory only
store.sync()                         // write state → URL
store.subscribe(s => console.log(s)) // notified on set / reset / back-forward
```
