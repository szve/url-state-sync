# 开始使用

`url-state-sync` 是一个极简、类型安全的 store:状态存在内存里,并同步到 URL query ——
**由你决定何时写入,而且值会保持声明的类型。**

## 安装

```bash
pnpm add url-state-sync
```

## 用法

```ts
import { createURLStore } from 'url-state-sync'

const store = createURLStore({
  keyword: '',
  page: 1,
  tags: [] as string[],
})

store.set({ keyword: 'hello' })      // 只改内存
store.sync()                         // 把 state 写入 URL
store.subscribe(s => console.log(s)) // set / reset / 后退前进 时通知
```
